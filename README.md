
# DND React Todo App

A todo app that includes a clean ui and dnd feature.
## Features

- Drag and drop
- Clean UI

  
## Bilgisayarınızda Çalıştırın

Clone Project

```bash
  git clone https://github.com/atakde/react-dnd-todo
```

Go to project folder

```bash
  cd react-dnd-todo
```

Install packages

```bash
  npm install
```

Run project

```bash
  npm run start
```
